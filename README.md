# Projects
- API para criar meus projetos que estão no meu portifólio.

## Techs
- Node.js
- express
- knex
- Typescript
- Postman

## Rotas do projeto
- Métodos Get: Pegar todos os projetos, projeto pelo id.
- Métodos Post: Criar projeto.

## Inicializar projeto
```bash
# Install packages
$ npm install

# Migrations
$ npm run migrations

# Run Back-end
$ npm run dev
```
